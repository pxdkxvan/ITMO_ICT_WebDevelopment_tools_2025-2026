# Техническая документация проекта

Этот набор файлов описывает текущую реализацию сервиса `Personal Finance API` в репозитории.

- `01_architecture.md` — архитектура приложения, состав модулей и общий поток запроса.
- `02_database.md` — модель данных, ограничения, связи и полная Mermaid-диаграмма БД.
- `03_http_api.md` — полный каталог HTTP-эндпоинтов с параметрами, примерами и ошибками.
- `04_business_flows.md` — где живет бизнес-логика, цепочки вызовов и ключевые сценарии.
- `05_change_navigation.md` — практическая навигация: что менять для эндпоинтов, логики и БД.

Ключевая точка входа приложения: `main.py` -> `app/main.py`.
